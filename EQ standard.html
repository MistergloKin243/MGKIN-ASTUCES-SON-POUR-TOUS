🔥 10 Conseils de Mixage

1. Bon gain staging

Évite la saturation dès le départ en gardant un signal bien équilibré.

2. Écoute à faible volume

Un bon mix doit rester clair même bas.

3. EQ pour corriger

Coupe d’abord les fréquences inutiles avant d’ajouter.

4. Compression = contrôle

Stabilise la dynamique au lieu d’écraser le son.

5. Panoramique

Aère ton mix en plaçant intelligemment gauche/droite.

6. Réverbération

Trop de reverb = mix noyé. Utilise-la avec parcimonie.

7. Tracks de référence

Compare ton mix à un morceau pro du même style.

8. Pauses régulières

Repose tes oreilles pour garder un jugement frais.

9. Mix en mono

Si ça sonne bien en mono, ça brillera en stéréo.

10. Imperfections = vie

Un mix trop parfait peut sembler froid.

🎛️ Réglages EQ Standard (Points de départ)

🎤 Voix Lead

Coupe bas : 80–100 Hz
Présence : 2–5 kHz
Air : 10–12 kHz
Astuce : surveille les sifflantes (5–8 kHz).

🎤 Back Vocals

Coupe bas : 100 Hz
Médium : 200–400 Hz (réduire)
Présence : 3–5 kHz
Astuce : EQ plus fin que la lead.

🎸 Guitare acoustique

Coupe bas : 80 Hz
Brillance : 5–8 kHz
Nasalité : 800 Hz–1.2 kHz (réduire)

🎸 Guitare électrique

Coupe bas : 80–100 Hz
Boue : 200–300 Hz (réduire)
Présence : 3–5 kHz

🎹 Piano acoustique

Coupe bas : 50–80 Hz
Clarté : 2–5 kHz
Air : 10–12 kHz
Astuce : réduire 300–500 Hz si “boxy”.

🎹 Electric Piano

Coupe bas : 60–80 Hz
Chaleur : 200–300 Hz (+)
Clarté : 3–5 kHz

🎻 Strings / Pads

Coupe bas : 100 Hz
Corps : 200–400 Hz
Air : 8–12 kHz

🎸 Basse

Sub : 40–80 Hz
Corps : 100–200 Hz
Clarté : 700 Hz–1.2 kHz
Définition : 2–4 kHz

🥁 Kick

Sub : 50–80 Hz
Boue : 200–300 Hz (réduire)
Click : 2–4 kHz

🥁 Snare

Corps : 150–250 Hz
Punch : 1–2 kHz
Clarté : 5–8 kHz

🥁 Hi-Hat

Coupe bas : 200–300 Hz
Brillance : 8–12 kHz
Astuce : attention aux sifflantes (6–8 kHz).

🥁 Toms

Corps : 80–120 Hz (floor), 100–200 Hz (rack)
Boue : 300–400 Hz (réduire)
Attaque : 4–6 kHz

🥁 Overheads

Coupe bas : 100 Hz
Clarté : 3–5 kHz
Air : 10–12 kHz
